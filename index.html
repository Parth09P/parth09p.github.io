Skills : Python, FastAPI, Docker, Azure Cloud, HuggingFace Transformers, PyTorch, NLP, Git, PyTest, Pylint, SonarQube

An AI assisted workbench designed to help scientists analyze the data produced during pre-clinical vaccine trials and also assist them to publish those findings effectively.

  • Implemented a micro-service for summarizing historical reports using the state-of-the-art Transformer-based techniques.
  • Created a corpus using unstructured documents as a baseline for training a language model, thereby reducing the need for annotated data.

  • Implemented scoring metrics for evaluating the performance of different models on reference texts and generated summaries.
  • Designed and implemented a feedback mechanism to take in the feedback of model's performance and further help the model improve down the line.

  • Used OrientDB, a graph based database for communicating with the UI, and AWS S3 as a cloud storage for files processed on AWS EC2.
